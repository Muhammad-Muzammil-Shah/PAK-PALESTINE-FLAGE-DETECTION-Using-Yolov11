# PAK-PALESTINE Flag Detection Using YOLOv11

This project demonstrates flag detection for Pakistan and Palestine using the YOLOv11 object detection model. The Jupyter notebook walks through training, inference, and evaluation steps for flag detection in images.

   ```bash
   pip install numpy pandas opencv-python torch yolov11
   ```
